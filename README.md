# Desktop Fox
Heya!
This project is a little desktop pet fox! It'll walk around your screen while playing a game or watching a youtube video.
I'm using this project as a way to learn python and it's gui editor.

It has 4 main behaviors
-Walking
-Sitting
-Dragged (Picking up w/ Mouse)
-Following Ball

To install : 
Download dist.zip and unpack.
From there, navigate into the FoxMain folder, and there will be an exe.
